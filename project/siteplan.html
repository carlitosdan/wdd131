Site Name

The site is named incasoft.net, I focused on creating a website for offering some services based in web environment. The logo and brand was created some years ago. The idea is use the knowledge for create a tailored website for accomplish our achievements.

Site Purpose

The purpose is offer web services such as: modern, responsive and functional designs for a complete website, web development for small and medium size companies, hosting and domain name. The site have 3 pages:

  • Home A review about our experience, client testimonials, our work process, the way how we work for accomplish 100% satisfaction of clients.
  • Services Description about three principal services, boxes with each of us.
  • Contact A form for email contacts, basic addresses information for contact us.
